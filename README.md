DFS Visualization
The Depth First Search (DFS) algorithm is a method for traversing or searching tree and graph data structures by exploring as far as possible along a single branch before backtracking. 

My program visually shows in the console the algorithm of passing the DFS on the map in search of an exit from the start point. If the path is not found, it returns to the last point of divergence and continues the path from there. When an exit is found, the program terminates.
